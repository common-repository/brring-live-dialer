== Frequently Asked Questions ==

= Are there any hidden charges? =

Brring Live Dialer has no hidden charges. Signup and receive $5.00 non-expiring call credit, completely free. No monthly recurring charges. If you receive no calls through Brring you don't pay a penny.

= What does Brring let me do? =

Brring adds a call widget to your site which visitors can use to connect with you by calling or scheduling a callback. You can use it to connect with leads or support your customers.

= Do I have to install any software on my server to get this working? =

Not at all! Brring is a cloud based service. Simply configure the plugin and you're done!

= Which web browsers work best with this plugin =

Though designed to work on most browsers, Brring works best in the following environment: Google Chrome (PC, Mac), Microsoft Edge (PC), Firefox, Safari and iOS and Android.

= Is it free to use? =

The plugin is completely free. If you have a Brring account setup with Live Dialer you can use the plugin with no restrictions.

== Usage ==

After enabling the plug in, head on to the Live Dialer settings page to change customize its look and feel to match your site. When done, enable it by visiting the account configuration page and completing the instant signup process.

== Installation ==

*Server Requirements:* PHP5 or PHP7.

*Wordpress versions:* Wordpress 3.4 and up.

Note: You will need a Brring account. <a href="https://www.brring.com/?ref=wordpress">Create one right here and get $5.00 free call credit to get you going</a>.

Step-by-step Guide:

* Install plugin from WordPress directory and activate it.
* Under the Settings section, click on Brring to add your Live Dialer Brring embed code.
* Customize the call widget to your preference in your <a href="https://app.brring.com/account/profile/live-dialer">Brring Live Dialer settings page</a>.
* All future changes can be made from the Brring Live Dialer settings page.
